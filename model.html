Models

gpt-oss:120b

An open-weight 120B parameter model with strong reasoning and tool-use abilities.

ollama run gpt-oss:120b

This model is designed for general-purpose reasoning, code generation, and tool use. It runs locally with ollama run, and is also available on Ollama Cloud for users without the hardware to host it.

Examples

ollama run gpt-oss:120b "Explain quantum entanglement to a high schooler."

API

curl http://localhost:11434/api/chat -d '{
  "model": "gpt-oss:120b",
  "messages": [{ "role": "user", "content": "Hello" }]
}'
